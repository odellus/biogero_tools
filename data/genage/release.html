GenAge Build 20: Release Notes

Release date: February 9, 2020
Number of genes in model organisms: 2,205
Number of human genes: 307

Build 20 is a significant update in terms of content. The set of human ageing genes has remained the same but existing entries have been further curated, resulting in the addition of several bibliographic references. Observations concerning many genes were also updated to reflect new findings.

The model organismsâ€™ set has been significantly extended with over 50 new genes associated with longevity added. New references and observations were also added to existing entries.

Please remember these notes refer to the latest stable build of GenAge. The current GenAge stats may differ slightly from the above due to improvements after the build's release.
